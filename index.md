# MegaForm for DNN (DotNetNuke)

Build forms, capture submissions, read & write SQL tables, and consume submission
data with Razor — all inside your **DNN** portal.

This site covers **MegaForm on DNN specifically**. For the Oqtane edition see the
[Oqtane documentation](https://cisssolution.github.io/MegaformDocs/).

## Start here

- **[Set up the MegaForm module](articles/dnn-module-setup.md)** — add MegaForm to an admin page, switch it to Dashboard mode, toggle Windowed ⇄ Full Screen.
- **[Read & write a SQL table](articles/dnn-sql-table.md)** — point MegaForm at a real database table and build a form from its columns.
- **[Cascade-SQL dropdowns](articles/cascade-sql-dropdowns.md)** — parent → child dropdowns that read from a SQL table.
- **[Display submissions with Razor](articles/dnn-razor-display-submissions.md)** — Card, ListView and detail layouts over your form data, with copy-paste Razor.

## Programming (SDK)

The MegaForm SDK is identical on DNN and Oqtane — read submissions, write data, render forms and download files from your own code:
